💡 دليل إضافة العروض والبكجات

كيفية إضافة العروض والبكجات

  1. إنشاء الفئة: تأكد من وجود فئة "الخصومات والعروض" (أو يمكنك إضافة فئة جديدة باسم discount من قسم الفئات).
  2. إضافة عرض (Offer):
    • انتقل إلى "عناصر القائمة" واضغط على "صنف جديد".
    • اكتب اسم العرض (مثال: "عرض وجبة الغداء").
    • اختر فئة "الخصومات والعروض".
    • ضع السعر المخفض في "السعر الأساسي".
    • اكتب تفاصيل العرض والخصم في حقل الوصف.
  3. إضافة بكج (Package):
    • اضغط على "صنف جديد" واكتب اسم البكج (مثال: "بكج العائلة الصحي").
    • في حقل الوصف، اكتب محتويات البكج (مثال: 3 وجبات رئيسية + 3 مشروبات).
    • لتحديد خيارات/أحجام مختلفة للبكج، استخدم قسم الكميات والأحجام (Portions) بالأسفل (مثال: إضافة حجم "بكج لشخصين" بسعر 80 ر.س، وحجم "بكج لـ 4 أشخاص" بسعر 150 ر.س).

How to Add Offers & Packages

  1. Create the Category: Ensure the "Discounts & Offers" category is active (or add a new one with ID discount from the Categories section).
  2. Adding an Offer:
    • Go to "Menu Items" and click "New Item".
    • Enter the offer name (e.g. "Lunch Special Offer").
    • Select the "Discounts & Offers" category.
    • Set the discounted price in "Base Price".
    • Explain the discount details in the description fields.
  3. Adding a Package (Bakkage):
    • Click "New Item" and enter the package name (e.g. "Family Healthy Package").
    • In the description fields, list the package items (e.g. 3 Main Meals + 3 Drinks).
    • To specify different package configurations, use the Portions section at the bottom (e.g. add "Package for 2" at 80 SAR, and "Package for 4" at 150 SAR).
